Class: Lottotron

Lottotron(maxNumber) → {Lottotron|Error}

Создает объект, предназначенный для выдачи в произвольном порядке не повторяющихся натуральных чисел. Числа выдаются из диапазона от нуля до заданного пользователем значения включительно.
